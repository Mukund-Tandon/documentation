# Facet search

The `/facet-search` route allows you to search facet values. Facet search supports prefix search and typo tolerance. The returned hits are sorted lexicographically in ascending order

## Perform a facet search

Search facet values.

<RouteHighlighter method="POST" route="/indexes/{index_uid}/facet-search"/>

<Capsule intent="warning">
Multi-word facets are not supported in some cases. Suppose your dataset has an `author` field, searching for `Jane` will return `Jane Austen`, but searching for `Austen` will not return `Jane Austen`.
</Capsule>

### Body

| Name                                                          | Type             | Default value | Description                                         |
| :------------------------------------------------------------ | :--------------- | :------------ | :-------------------------------------------------- |
| `facetQuery`                                                  | String           | `null`        | Search query for a given facet value                |
| `facetName`                                                   | String           | `null`        | Facet in which to search for `facetQuery`           |
| [`q`](/reference/api/search#q)                                | String           | `""`          | Query string                                        |
| [`filter`](/reference/api/search#filter)                      | Array of strings | `null`        | Filter queries by an attribute's value              |
| [`matchingStrategy`](/reference/api/search#matching-strategy) | String           | `last`        | Strategy used to match query terms within documents |

### Response

| Name                   | Type             | Description                                                             |
| :--------------------- | :--------------- | :---------------------------------------------------------------------- |
| **`facetHits`**        | Array of objects | Array of objects for the facet search. Each object represents one facet |
| **`facetQuery`**       | String           | The original `facetQuery`                                               |
| **`processingTimeMs`** | Number           | Processing time of the query                                            |

#### `facetHits` array

| Name        | Type    | Description                           |
| :---------- | :------ | :------------------------------------ |
| **`value`** | String  | Facet value matching the `facetQuery` |
| **`count`** | Integer | Count of the value in the documents   |
